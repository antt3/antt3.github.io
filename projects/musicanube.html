MusicaNube

Live Site Github Repo

MusicaNube is a SoundCloud inspired site featuring Songs, Comments, Playlists, Search, User Profiles, and Continuous Play. MusicaNube uses a Javascypt Express.js backend, a PostgreSQL database, and a JavaScrypt React/Redux frontend.



Solo Project using JavaScrypt, Express.js, React.js, Redux, PostgreSQL, CSS, and HTML
